# YoloPose

Gradio Demo for YOLO Pose Estimation models. Detect and predict the poses of people in images, videos, or via webcam. Supports YOLOv8, YOLO11, and YOLO26 pose models.

See [Ultralytics GitHub](https://github.com/ultralytics/ultralytics) | [Models Page](https://docs.ultralytics.com/models/) for more details.
